Technology targets

Sosol's end goal is targeting a completely opensource, censorship resistant platform deployment. But the token will also have utility to be used on every social media platform no matter how centralised. After alpha, we'll be releasing a road map outlining the timeline from centralised delivery in alpha, to a completely decentralised blockchain platform.

Development progress

Alpha is being built out as we speak and a lot of the Solana community on Twitter are following harkl_ ready to be alpha testers. There are 3 parts of the application: the React client, The token, and the GraphQL API. For alpha launch they'll be deployed on a centralised host, but the overall goal on the roadmap is to open source the client and move the GraphQL API into Orbit DB built with IPFS.
